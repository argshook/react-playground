# React playground

## How to do

1. `npm i -g babel-cli babel webpack mocha`
1. `npm i -g webpack-dev-server`
1. `npm i`

## How

1. `npm start` - for dev (server at [http://localhost:8080/](localhost:8080))
1. `npm run build` - for prod

## test

1. `npm test` - once
1. `npm run test:watch` - guess what
